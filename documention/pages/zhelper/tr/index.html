7. zHelper

# zHelper Nedir? Ne işe yarar?

Projelerimizi geliştirirken bizim yerimize gerekli işleri yapan bir asistandır, Komut İstemi üstünde kullanılan bu asistan bizim yerimize şu işleri yapar.

# Oluşturabileceğimiz bileşenler

Kontrolcü: 
                    > php zhelper make controller Test\TestController          || Varsayılan Kontrolcü
                    > php zhelper make controller Test\TestController resource || Yönlendiriciler için hazır oluşturulabilir Kontrolcü
Model: 
> php zhelper make model Test\Test
Gözlemci: 
> php zhelper make observer Test\TestObserver
Ara katman yazılımı: 
> php zhelper make middleware Test\Test
Veritabanı Şablonu: 
> php zhelper make migration Users
Veritabanı Besleyicisi: 
> php zhelper make migration UsersSeeder

# Veritabanı Şablonu kullanımı

# Şablonu veritabanına aktarır

> php zhelper db migrate
Kolon yoksa kolonu ekler varsa da kolonu günceller. (eğer özelliklerini değiştirdiyseniz güncellemeden sonra gözükür)

# Şablonu olan bütün tabloları siler ve baştan yazar

> php zhelper db migrate fresh
Taze bir veritabanı.

# Veritabanı Besleyici kullanımı

Veritabanını beslemek için kullanılır, bunun anlamı (Sahte / Sürekli doldurulması gereken) verileri otomatik yazar. 
> php zhelper db seed

# Ön bellekleri temizlemek

> php zhelper cache clear sessions|caches|views
