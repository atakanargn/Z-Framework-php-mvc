4. Kontrolcüler

# Bir Kontrolcü neye benzer?

                class ... {
                    public function __construct() {
                        echo "Merhaba, bu bir __construct'tur";
                        $this->user = new User;
                    }
                    
                    public function index() {
                        $selam = 'Selam';                                    # resource/views/main.php Kalıp yolu
                        return View::view('home.index', compact('selam'), 'main');
                    }
                    
                    public function show($id) {
                        return view('home.show', ['user' => $this->user->first()], 'main');
                    }
                }
Kontrolcümüzde belirlediğimiz methodları Yönlendiricilerimizin içinde kullanabiliriz ve ayrıca kontrolcülerimizin de içlerinde kendi methodlarını kullanabiliriz.
Tabii ki bu yapıya sabit kalmak içindir yoksa istediğiniz gibi her şeyi her Kontrolcüyü her yerde de çağırabilirsiniz.
