3. Arayüzler
# Arayüzler nedir?
Arayüzler kullanıcıya yansıtılacak son görünüştür, ve yapımızın en önemli kısmı sayılabilir.

# Nasıl Kullanılır?

                # Bunun gibi
                echo view('home.index', ['hi' => 'hey'], 'main');
                
                # Bunun gibi
                echo zFramework\Core\View::view('home.index', ['hi' => 'hey'], 'main');
Kullanabiliriz.

# Zaten bir Arayüz içerisindeyken nasıl tekrar içeriye nasıl bir arayüz daha çağırabiliriz?

Arayüz içerisinde yeni bir arayüz çağırırken içerisinde olduğumuz arayüzün parametrelerini diğerine aktarabiliriz.
Bunu depoladığımız parametre $view_parameters parametresidir.
                <div>
                    Liste:
                    <?= view('home.list', $view_parameters) ?>
                    <?= View::view('home.list', $view_parameters) ?>
                </div>
Şeklinde çağırabiliriz.

# Belirli bir kalbı nasıl sürekli kullanırız?

Bu biraz detaylı bir konu olduğu için biraz uzun anlatacağım.
Yapının ağaç görünümü. 
resource
├─── views
│   ├───main.php
│   └───user.php

# main.php arayüzünün içeriği:

Kalıp sayfasında kalıbın ortaya yazılacak kısmında kesinlikle ve kesinlikle <!--body--> yazmak zorundadır. 
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Kalıp sayfası <?= $title ?></title>
                    </head>
                    <body>
                        <!--body-->
                    </body>
                    </html>

# user.php arayüzünün içeriği:

                    

                        zFramework!

# Arayüzümüzü Kalıbıyla birlikte çağıralım

                    function ...() {
                        return view('user', ['title' => 'Kullanıcılar Sayfası'], 'main');
                    }
Ve çıktımız şu şekilde olacaktır:
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="UTF-8">
                        <meta http-equiv="X-UA-Compatible" content="IE=edge">
                        <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        <title>Kalıp sayfası Kullanıcılar Sayfası</title>
                    </head>
                    <body>
                        <div>
                            zFramework!
                        </div>
                    </body>
                    </html>
Ve bu konuyu geçebiliriz.
