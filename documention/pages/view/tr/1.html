3.1. Arayüz Destekcisi

# Arayüz destekcisi ne işe yarar?

Arayüz destekçileri bir Arayüz'e sürekli vermemiz gereken parametreler yükünden bizi kurtarırlar.
Buna bir örnek verecek olursak:
# Yol: App\Providers\ViewProvider.php

class ViewProvider
{
    public function __construct()
    {
        View::bind('ev.kullanicilar', function () {
            $user = new User;
            return [
                'users' => $user->get()
            ];
        });
    }
}

# `ev.kullanicilar` yolundaki Arayüz sürekli `$users` parametresine sahip olacaktır.
