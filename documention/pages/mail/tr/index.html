6. E-posta

# E-Posta işlemlerimizi nasıl yönetiriz?

İlk önce bu özelliği kullanmadan bir kaç ayar yapmamız gerekiyor. 
config
├─── mail.php
Dosyasının içinde 
                    return [
                        'sending' => true,

                        'debug' => false,
                        'SMTPAuth' => true, || SMPT girişi
                        
                        'mail' => 'smtp.mailtrap.io',   || SMTP Giriş: Mail Sunucusu
                        'port' => 2525,                 || SMTP Giriş: Mail Sunucu Portu
                        'username' => '65a52bb05478e1', || SMTP Giriş: Mail Sunucu Kullanıcı Adı
                        'password' => 'd6b2953732bb0c', || SMTP Giriş: Mail Sunucu Kullanıcı Şifresi

                        'from' => ['Kimden gittiği bilgisi', 'from@example.com'],
                        'reply' => ['Kime dönüt yapılabileceği bilgisi', 'replyto@example.com']
                    ];
Gibi bir yapı ile karşılaşacağız.
Not: Eğer sending parametremiz true olmazsa gönderim yapamayız.
Bu bilgileri kendi bilgilerimiz ile doldurduktan sonra yapabileceğimiz işlemlere geçelim.

# Nasıl Mail gönderilir?

                Mail::to('mustafaomereser@gmail.com')->send([
                    'subject' => 'Mail başlığımız', || Bu kısımda göndereceğimiz mail`in başlığını ayarlamış oluyoruz.
                    'message' => 'test mesaj',      || isterseniz bu kısımda bir `Arayüz` de gönderebilirsiniz. Örnek: view('e-postalar.sifremi-unuttum', ['token' => Str::rand()])
                    'altbody' => 'Alt body',        || (Opsiyonel) Alt Body yazımız.
                    'attachements' => [             || (Opsiyonel) Ve istersek beraberinde dosya da gönderebiliriz.
                        'uploads/1.png',
                        'uploads/2.png'
                    ]
                ]);
Bu şekilde işlemlerimiz tamamlanmış olacaktır ve yukarıdaki örnek gibi artık E-posta gönderebileceğiz.
