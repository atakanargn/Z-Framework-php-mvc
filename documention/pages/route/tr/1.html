1.1. Form Örnekleri

# Form Örnekleri

* Resource bir controllerimizin olduğunu varsayıyorum, bu demek değil ki Resource olmayan bir yapıda kullanılamaz.
# Örnek Controller 
                    class UserController extends Controller
                    {
                        public function index()
                        {
                            return "Kullanıcılar liste sayfası";
                        }

                        public function show($id) { ... }

                        public function create() { ... }

                        public function edit($id) { ... }

                        public function store()
                        {
                            return "Kullanıcı Kayıt İşlemleri"
                        }

                        public function update($id)
                        {
                            return "Kullanıcı Güncelleme İşlemleri"
                        }

                        public function delete($id)
                        {
                            return "Kullanıcı Silme İşlemleri"
                        }
                    }
Bu Form Resource Controller içindeki store() methoduna gitmektedir. formda action özelliğini kullanmamın amacı daha iyi bir anlatış biçmi içindir. 
                    

                         (isteğe bağlı olarak direkt input yerine `` de kullanılabilir.)
Bu Form Resource Controller içindeki update() methoduna gitmektedir. formda action özelliğini kullanmamın amacı daha iyi bir anlatış biçmi içindir. 
                    <form action="<?= route('users.update', ['id' => 1]) ?>" method="POST">
                        <?= csrf() ?>
                        <?= inputMethod('PATCH') ?>
                        <input type="submit">
                    </form>
Bu Form Resource Controller içindeki delete() methoduna gitmektedir. formda action özelliğini kullanmamın amacı daha iyi bir anlatış biçmi içindir. 
                    <form action="<?= route('users.delete', ['id' => 1]) ?>" method="POST">
                        <?= Csrf::csrf() ?>
                        <?= inputMethod('DELETE') ?>
                        <input type="submit">
                    </form>
