16. API (Application Programming Interface, Uygulama Programlama Arayüzü)
Açıklama; API bize Arayüz olmadan sistemden veri almamıza yaramaktadır.

# Kullanım

Dosya yolu: route/api.php 
                Route::get('/test', function () {
                    echo "API Page / user_id: " . Auth::id();
                });
Örnek: http://localhost/api/test?user_token=12345678 (Eğer token geçerliyse kullanıcı giriş yapmış olacaktır.)
