8. CSRF
Saldırgan web siteleri veya saldırgan kişiler Web sitenize dışarıdan istek atabilirler.
Bu istekleri engellemenin en iyi yolu CSRF Token'dir.
CSRF Token GET harici tüm methodlarda spesifik olarak engellenmediği sürece gereklidir.
CSRF Token form örneklerine bakmak için buraya tıklayınız.
CSRF Token varsayılan olarak 10 dakikada bir yenilenmektedir.

# Hazır CSRF Token input almak

                Csrf::csrf();
                # Veya
                csrf();

# CSRF Token'i direkt almak

                Csrf::get();

# Rastgele yeni bir CSRF Token oluşturmak

                Csrf::set();

# CSRF Token'i kaldırmak

                Csrf::unset();

# CSRF Token'in yenilenmesine kalan zaman.

                Csrf::remainTimeOut();
