2.4. Besleyiciler
Açıklama; Besleyicilerin görevi boş/taze bir veritabanına rastgele test verileri girmektir. Veya her veritabanı sıfırlaması sonrasında girmek zorunda olduğunuz verileri bu kısımda tanımlayabilirsiniz.
Bir Besleyici nasıl oluşturulur ve çalıştırılır öğrenmek için zHelper buraya tıklayabilirsiniz.
* Bir Besleyici Örneği: 
                class Seeder
                {
                    public function __construct()
                    {
                        $this->user = new User;
                    }
            
                    public function seed()
                    {
                        $this->user->insert([
                            'username' => 'admin',
                            'password' => Crypter::encode('admin'),
                            'email' => 'admin@localhost.com',
                            'api_token' => Str::rand(60)
                        ]); 
                    }
                }
